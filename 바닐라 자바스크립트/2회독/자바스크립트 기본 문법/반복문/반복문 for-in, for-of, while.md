## for-in

for-in문은 데이터 타입 중 배열뿐만 아니라 Object에도 사용할 수 있는 반복문이다.

for (const key in object) {
// 실행할 코드 블록
}

## for-of

Array, Map, String 등 반복 가능한 객체에서 사용 가능한 반복문이다.

for (const key of iterable) {
// 실행할 코드 블록
}

## while

조건식이 true인 동안 코드 블록을 실행한다.

while (condition) {
// 코드 블록
}
