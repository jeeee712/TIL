## DOM 이벤트

웹 페이지를 이용할 때에는 웹 화면과 사용자 간에 수많은 상호작용이 이루어진다. 이렇게 사용자가 웹 페이지를 이용하면서 행하는 액션을 이벤트라고 한다.

## click이벤트(onclick)

클릭 이벤트는 사용자가 웹 페이지 안에서 마우스 왼쪽 버튼을 누를 때 발생하는 이벤트이다.

## change 이벤트(onchange)

값이 변경되었을 때 발생하는 이벤트이다.

## focus 이벤트(onfocus)

사용자가 텍스트, 숫자 같은 데이터를 입력할 수 있는 DOM 요소에 마우스를 위치할 때 발생하는 이벤트이다.

## blur 이벤트(onblur)

blur 이벤트는 현재 마우스가 포커스 되어 있는 DOM 요소를 빠져나가려 할 때 발생하는 이벤트이다.

## key 이벤트(onkeydown, onkeypress, onkeyup)

key 이벤트는 사용자가 입력창에서 키보드를 입력할 때 발생한다.

## scroll 이벤트(onscroll)

사용자가 웹 페이지에서 마우스를 스크롤하면 발생하는 이벤트이다.

## touch 이벤트

스마트폰, 태블릿 같은 모바일이나 터치가 가능한 PC에서 화면을 터치하면 발생하는 이벤트이다.

## load 이벤트

웹 페이지의 모든 HTML DOM 요소가 로딩되면 발생하는 이벤트이다.

## unload 이벤트

브라우저 창을 닫거나 다른 웹 사이트로 이동하게 되었을 때 보안을 위해 사용하는 이벤트이다.
